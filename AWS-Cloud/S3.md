# AMAZON S3 – COMPLETE & EASY NOTES

## 1. WHAT IS AMAZON S3?

S3 = **Simple Storage Service**

Amazon S3 is:
- A **Cloud Storage Service**
- Used to store:
  - Files
  - Images
  - Videos
  - Backups
  - Website files
  - Big Data
  - Logs

S3 is:
✅ Highly secure  
✅ Highly durable (99.999999999%)  
✅ Scalable  
✅ Pay only for what you use  

---

## 2. REAL-WORLD USE OF S3

- Website hosting
- Data backup
- Image & video storage
- Mobile app storage
- ML/DL dataset storage
- Company logs
- Database backups
- Archive storage

---

## 3. S3 BASIC STRUCTURE

```

S3 → Bucket → Object (File)

```

| Term | Meaning |
|------|---------|
| Bucket | Like a Folder |
| Object | File inside Bucket |

Example:
```

Bucket: my-data-bucket
Object: photo.jpg, video.mp4, data.csv

```

---

## 4. S3 STORAGE CLASSES

| Storage Class | Use |
|----------------|-----|
| Standard | Frequent access files |
| Intelligent Tiering | Auto moves based on usage |
| Standard-IA | Less frequent data |
| One-Zone IA | Cheap + Single zone |
| Glacier | Archival storage |
| Glacier Deep Archive | Cheapest storage |

✅ Glacier is used for:
- Government records
- Old company files
- Legal backups

---

## 5. CREATING AN S3 BUCKET (STEP BY STEP)

Go to:
```

AWS Console → S3 → Create Bucket

```

### Step 1️⃣: Bucket Name
- Must be **unique globally**
Example:
```

my-first-s3-bucket-2025

```

---

### Step 2️⃣: Region
Choose:
```

ap-south-1 (Mumbai)

```

---

### Step 3️⃣: Object Ownership
```

ACLs Disabled (Recommended)

```
Only **Bucket Owner** has access.

---

### Step 4️⃣: Block Public Access
✅ Enabled by default  
❌ Disable ONLY if you want **public access**

---

### Step 5️⃣: Bucket Versioning
```

Disabled by default

```
✅ Used to keep multiple versions of same file.

---

### Step 6️⃣: Encryption
```

Enable Encryption → SSE-S3

```
Protects data at rest.

---

### ✅ Click: CREATE BUCKET

---

## 6. UPLOADING FILES TO S3

Path:
```

S3 → Select Bucket → Upload → Add Files → Upload

```

You can:
- Upload files
- Create folders
- Delete objects
- Download objects

---

## 7. FOLDER CREATION IN S3

Inside Bucket:
```

Create Folder → Enter Name → Create

```

Example:
```

images/
backup/
videos/

```

---

## 8. PUBLIC ACCESS TO FILES (STATIC WEBSITES / IMAGES)

By default:
❌ S3 blocks all public access

To allow public access:

### Step 1️⃣: Bucket Permissions
- Disable:
```

Block Public Access

```

---

### Step 2️⃣: Object Ownership
Enable:
```

ACLs Enabled

```

---

### Step 3️⃣: File Permission
Go to:
```

File → Permissions → ACL → Everyone → Read → Save

```

✅ Now file becomes **publicly accessible**

---

## 9. S3 VERSIONING

Versioning keeps:
- Old copies of files
- Protects from accidental delete

Example:

| Upload | Version |
|--------|---------|
| 1st Upload | Null |
| 2nd Upload | v2 |
| 3rd Upload | v3 |

### Enable Versioning:

```

Bucket → Properties → Versioning → Enable

```

| Permission | Meaning |
|-------------|----------|

- Upload

# ✅ S3 INTERVIEW QUESTIONS & ANSWERS

### 1. What is Amazon S3?
S3 is a cloud storage service for storing any type of data.

---

### 2. What is a Bucket?
A bucket is a container for storing files in S3.

---

### 3. What is an Object?
A file stored inside an S3 bucket.

---

### 4. Is S3 free?
No. It follows **pay-as-you-use** model.

---

### 5. What is S3 Versioning?
It keeps multiple versions of the same file.

---

### 6. Can we host a website on S3?
Yes. S3 supports static website hosting.

---

### 7. What is Glacier?
Low-cost archival storage for long-term backup.

---

### 8. Why is S3 secure?
Because it supports:
- Encryption
- IAM
- Bucket policies
- ACLs

---

### 9. What is Bucket Policy?
Rules that control bucket access.

---

### 10. Difference between S3 and EC2?

| S3 | EC2 |
|-----|------|
| Storage Service | Compute Service |
| Stores files | Runs applications |
| No OS | Has OS |

---

### 11. What is IAM role used in S3?
To securely allow EC2 to access S3.

---

### 12. What happens if versioning is enabled and file is deleted?
Only the latest version is removed, old versions remain.

---

### 13. What is Object Lock?
Prevents file deletion for a fixed time.

---

### 14. What is S3 Lifecycle?
Automatically moves files across storage classes.

---

### 15. Can S3 store unlimited data?
Yes. It is **infinitely scalable**.
